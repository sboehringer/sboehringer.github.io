About

ColoredChromosomes is a software project to automate complex ideogram drawing tasks. Using this program you can draw complex ideograms and add various annotations. Via a postscript intermediate format images of any desired quality can be produced (see e.g. GIMP or ImageMagick).

Current version: 1.4.2pre2

The sister project Loci Locations allows to retrieve physical positions of loci from certain databases.
