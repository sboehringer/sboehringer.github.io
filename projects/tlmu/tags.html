\meta{key;value}

Sets a meta key word in the meta information of the document. Currently supported keys are 'title', 'author', 'keywords'.

\comment{some comment}

Add a comment, or wrap a table (see table).

\section{section name} ... \endSection

Write a text section with a heading.

... \

Include text that is included in the final document in a way that closely resembles its text editor representation.
