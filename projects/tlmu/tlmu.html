The Tex Like MarkUp language (TLMU) specification

Each tlmu document is a plain ASCII file.

Grammar

The grammar of the file is as follows: 

	Document :=   Text Sequence
				| Sequence
				| Markup

	Sequence :=   Markup Text
				| Sequence Markup Text

	Markup :=	'\' AlphaNum
				| '\' AlphaNum '{' Text '}'
